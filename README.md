# QuoteCraft 外贸报价工作台

一个纯静态的外贸报价单工具，适合工业品、自动化配件、小工厂和跨境 B2B 销售使用。

## 功能

- 实时生成英文商业报价单
- 自动计算报价总额、预估毛利和毛利率
- 自动生成英文跟进邮件
- 支持复制报价文本和邮件
- 支持浏览器打印或另存为 PDF
- 完全本地运行，不需要后端和数据库

## 本地预览

直接双击 `index.html` 即可打开。

如果想用本地服务预览：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/index.html
```

## 部署

这是纯静态网站，部署时只需要上传 `index.html`。

推荐方式：

1. GitHub Pages：把 `index.html` 放到仓库根目录，在仓库设置里开启 Pages。
2. Netlify：拖拽整个文件夹到 Netlify Drop。
3. Vercel：导入仓库，保持默认静态部署设置。
4. Cloudflare Pages：导入仓库，构建命令留空，输出目录设为 `/`。

## 商业化方向

- 公司 Logo 和品牌色
